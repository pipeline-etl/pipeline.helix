# Pipeline.Helix

Pipeline.Helix is a content diffing algorithm for pipeline imports.

Installation
------------

* Install Pipeline.Helix with Composer or with your own installer.
* Pipeline.Helix follows the [semantic versioning][2] standards.

Community
---------

* Follow us on [GitHub][3].

Contributing
------------

Pipeline.Helix is an Open Source, community-driven project. Join by contributing code or documentation.

If you encounter any issues when using Pipeline.Helix you can report them [on github][4]
